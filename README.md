# Candidate Success Prediction

## Overview

This repository is part of a **Candidate Success Prediction Hackathon** project. It contains the complete data analysis and preprocessing pipeline developed to prepare the dataset for machine learning.

## My Contribution

I was responsible for the data analysis phase of the project, which includes:

* Data collection and preprocessing
* JSON to CSV conversion
* Data cleaning and handling missing values
* Exploratory Data Analysis (EDA)
* Feature engineering
* Data encoding and transformations
* Preparing the final datasets for model development

## Machine Learning Model

The machine learning model will be developed separately by another team member. Once completed, the trained model, source code, and documentation will be added to this repository with proper contributor names and credits.

## Dataset

The dataset files are hosted on Google Drive because they exceed GitHub's file size limit.

**Dataset Download:** <Google Drive Link>

After downloading, place all CSV files inside the `data/` directory before running the notebooks.
